# DFCirrus
Sky background modelling for inidividual frames with Galactic Cirrus taken by Dragonfly

### Dependencies

- matplotlib
- numpy
- scipy>=1.6.0
- astropy>=4.0
- photutils>=0.7.2
- scikit-learn>=0.24.2
- scikit-image>=0.19.2
- reproject>0.9
- tqdm
- joblib

```bash
conda install -c conda-forge openjdk
```

### Installation
---

```bash
cd <install directory>
git clone https://github.com/NGC4676/DFCirrus.git
cd DFCirrus
pip install -e .
```
